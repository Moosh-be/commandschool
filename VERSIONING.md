# Convention de versionnement

Format : `MAJOR.MINOR.PATCH[-platform]`

- **MAJOR** : Changement cassant la rétrocompatibilité
- **MINOR** : Ajout de fonctionnalités
- **PATCH** : Corrections de bugs, améliorations techniques
- **platform** : Suffixe identifiant la plateforme cible (`-win`, `-osx`, `-linux`, `-nas`, `-raspi`, ...)

Exemples : `0.1.2-win`, `0.2.0-osx`, `1.0.0-win`
