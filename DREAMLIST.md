# Dreamlist

## Fonctionnalités

- Pagination avec barre de progression (comme un lecteur terminal)
- Raccourci pour exécuter la commande directement dans le terminal de l'utilisateur
- Mode "défi" — deviner la catégorie ou la commande avant de révéler
- Mode "exploration guidée" — parcours par catégorie
- Découverte du jour automatique (rotation basée sur la date)
- Mode "familier" — mémoriser les découvertes vues et exclure

## Améliorations UX

- Terminal plus aéré, interlignes plus grands
- Animations de chargement (barre de progression, spinners)
- Couleurs personnalisables
- Mode sombre / mode clair en console
- Copier-coller du résultat brut d'un clic

## Intégrations

- Export d'une découverte en Markdown/PDF
- Partage sur Mastodon/Twitter avec un texte pré-formaté
- Synchronisation de l'historique entre machines
- Plugin VS Code — `cmdschool` dans le terminal intégré
